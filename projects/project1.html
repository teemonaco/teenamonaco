E-Commerce Checkout Redesign

Overview

This project aimed to address high cart abandonment rates, improve the checkout process and integrate PayPal Pay Now and Apple Pay.

Before
After

Rersearch & Analysis

Device Category

Majority of our users are shopping on a mobile device.

User Demographic

Most of our users are ages 55+ which statistically struggle more with vision disabilities.

Page Abondonment Rate

Page Abandonment rates show that majority of users leave on the first page, 'My Cart'.

New Designs

I faced several challenges such as addressing accessibility issues, fixing legacy code, and ensuring WCAG compliance. Solutions included improving color contrast, enhancing user feedback loops, and optimizing for mobile responsiveness.

Disabled Button Styling Update

We used a disabled animation but had active styling on the disabled button. The button now clearly shows it's non-interactive.

PayPal Branding Compliance

Adjusted PayPal button styling to align with PayPal's branding, avoiding the use of gray when it was previously styled as active.

Error Messaging on CTA Buttons

Combined confusing error messages and added clearer call-to-action indicators for error resolution.

Enhanced Priority Message Display

Revised the layout for high-priority messages so essential information is highlighted and distinguishable.

Responsive Checkout Layout

Made inner content of the checkout page responsive, accommodating screen sizes down to mobile without breaking the layout.

Primary CTA Contrast Update

Adjusted primary CTA button colors to meet AA accessibility standards, ensuring sufficient contrast for all users.

Resolved Conflicting Messages

Streamlined messaging to eliminate conflicts, adding clarity to avoid misinterpretations.

Error Message Positioning

Moved error messages closer to the related input fields to improve usability and context for users.

Header Button Contrast Update

Enhanced the progress bar colors to meet AA contrast standards, ensuring better visibility and clearer stage indicators.

Progress Bar Update

Enhanced the progress bar colors to meet AA contrast standards, ensuring better visibility and clearer stage indicators.
